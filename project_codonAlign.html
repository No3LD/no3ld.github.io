Codon Alignment Tool

I completed this project for a Bioinformatics class in my senior year of my undergraduate degree. It is a python implementation of local alignment via the Smith-Waterman algorithm and global alignment via the Needleman-Wunsch algorithm that translates the user given nucleotide bases in a genetic sequence into its correct codon sequence before running alignment algorithms.
